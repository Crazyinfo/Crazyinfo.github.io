Hexo常用命令

2021-10-24 博客相关 Hexo 0 评论 字数统计: 287(字) 阅读时长: 1(分)

简写指令

命令 简写 说明
hexo new “第一篇文章” hexo n “第一篇文章” 新建一篇博客
hexo new draft “new draft” 新建一篇草稿博客
hexo publish hexo p
hexo sever hexo s Hexo 会监视文件变动并自动更新，除修改站点配置文件外,无须重启服务器,直接刷新网页即可生效
hexo deploy hexo d 将本地数据部署到远端服务器(如github)
hexo generate hexo g 生成静态网页 (执行 hexo g后会在站点根目录下生成public文件夹, hexo会将”/blog/source/“ 下面的.md后缀的文件编译为.html后缀的文件,存放在”/blog/public/ “ 路径下)
hexo deploy generate hexo deploy -g
hexo generate deploy hexo generate -d 作者：雪上行者_ 链接：https://www.jianshu.com/p/7b8faf77d1af 来源：简书 著作权归作者所有。商业转载请联系作者获得授权，非商业转载请注明出处。
hexo clean 无 清除缓存 ,网页正常情况下可以忽略此条命令,执行该指令后,会删掉站点根目录下的public文件夹
hexo init 文件名称 chu实话XX文件夹名称
